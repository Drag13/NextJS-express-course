Що таке Next.JS

NextJS - клієнт-серверний фреймворк для створення веб застосунків на React

Клієнт-серверний - дозволяє вам писати одночасно і фронт-енд і бекенд

Фреймворк - має додаткові інструменти та власну структуру проекту

Заснований на React - використовує React.JS для відображення контенту

Можливості Next.JS

Серверний рендеринг навіть без серверу

Можливість писати серверний код

Фокус на швидкості

Відмінність Next.JS та React

React це бібліотека для відображення клієнтської частини

Next.JS - клієнт-серверний фреймворк який використовую React для відображення

Next.JS =
React + сервер + роутер + багато іншого
