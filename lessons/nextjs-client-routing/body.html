Роутинг в Next.JS

Засновний на структурі вашого проекту

Як додавати нові сторінки

Створити теку, яка має називатися так як ви хочете щоб називався ваш роут

У створену теку покласти файл page.tsx | page.jsx

У файлі page створити react компонент і зробити дефолтний експорт

Динамічний роутинг

course/[pageId]/page.tsx

Як додати layout

- Додати файл layout.tsx, поруч зі сторінкою, яку ви бажаєте огорнути

- У файлі layout.tsx cтворити компонент, який приймає в якості props children та відмалювати переданий children
- Зробити дефолтний експорт створеного компоненту

Layout VS Template

Layout зберігає стейт та не перерендерюється під час зміни сторінки

Навіщо потрібні файли loading.tsx та error.tsx

Loading.tsx - показується коли ваша сторінка генерується та завантажується

Error.tsx - буде показаний у випадку, якщо під час генерації сторінки з'явиться помилка
